## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript Designli real challenge.

## Test inputs

eml directory contains EML file to validate and email-example.png to see the email template was used to test

## Postman 
postman directory contains real-challenge.postman_collection.json file to validate

## Running the app

```bash
# production with npm
$ npm install
$ npm run build
$ npm run start:prod
# docker container
$ docker build . -t email-parser-real-challenge
$ docker run -p 3001:3001 -d  email-parser-real-challenge
```
